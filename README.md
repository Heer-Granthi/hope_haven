\# Hope Haven



A PHP \& MySQL web application to connect donors with orphanages in Nanded district.



\## Features

\- View orphanages

\- Donate items/money

\- Book visits

\- Admin approval system



\## Setup Instructions

1\. Clone the repository

2\. Place project in XAMPP htdocs folder

3\. Start Apache \& MySQL

4\. Import `hope\_haven.sql` in phpMyAdmin

5\. Open http://localhost/hope\_haven



\## Technologies Used

\- PHP

\- MySQL

\- HTML/CSS

\- XAMPP
